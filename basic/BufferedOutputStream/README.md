### 2) Тема: Буферизованный вывод в файлы (BufferedOutputStream)

2) Пример:  
- **BufferedWriteText.java:** Запись текста в файл с буферизацией для повышения производительности.

3) Объяснение темы:  
Буферизованные потоки вывода оборачивают базовые потоки, такие как FileOutputStream, накапливая данные в временном буфере перед их физической записью на диск. Это снижает количество прямых обращений к аппаратным ресурсам, ускоряя операции для последовательных или мелких записей, и предотвращает фрагментацию при частых изменениях данных. Идеально для приложений, где важна эффективность, например, логирование или генерация отчетов.

4) Основные методы и особенности:  
- **write(int b) / write(byte[] b, int off, int len):** Запись байта или фрагмента массива в буфер; данные буферизуются до вызова flush() или close().  
- **flush():** Принудительная выгрузка буфера в базовый поток, обеспечивая timely запись без ожидания заполнения.  
- **close():** Закрытие потока с автоматическим flush(), освобождающее ресурсы; рекомендуется в try-with-resources.  
- Особенности: Размер буфера по умолчанию 8192 байт (настраивается конструктором); минимизирует системные вызовы; поддержка цепочек оберток (e.g., с DataOutputStream); обработка IOException для ошибок диска или прав доступа.
